# Contributing to Jujutsu Battlegrounds

Thank you for your interest in contributing to **Jujutsu Battlegrounds**! We'd love your help making this game even better.

## 🎯 How to Contribute

### 1. **Report Bugs**
Found a bug? Let us know by:
- Opening an **Issue** on GitHub
- Include steps to reproduce the bug
- Describe what you expected vs. what happened
- Add screenshots if possible

### 2. **Suggest Features**
Have a cool idea? Create an Issue with:
- Clear title: "Feature: [Your Idea]"
- Detailed description of the feature
- Why it would improve the game
- Any mockups or examples

### 3. **Submit Code**
Want to code? Follow these steps:

1. **Fork the repository**
2. **Create a branch** for your feature:
   ```
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes** with clear, descriptive commits
4. **Test thoroughly** before pushing
5. **Submit a Pull Request** with:
   - Clear description of changes
   - Why these changes improve the game
   - Any breaking changes noted

## 📋 Coding Standards

- Follow **Lua/Luau best practices**
- Use clear, descriptive variable and function names
- Add comments for complex logic
- Keep files organized in the existing structure
- Test your code in Roblox Studio before submitting

## 🎨 Areas We Need Help With

- **New Characters** - Create balanced character designs with unique movesets
- **Boss Battles** - Design challenging AI patterns and mechanics
- **UI/UX** - Improve mobile controls and visual design
- **Balance** - Help tune damage values and combo mechanics
- **Documentation** - Improve guides and tutorials
- **Sound/Effects** - Enhance audio and visual feedback

## 🚀 Getting Started Locally

1. Clone the repository:
   ```
   git clone https://github.com/donovanhamilton227-spec/Jujitsu-battlegrounds-.git
   ```

2. Open the project in **Roblox Studio**

3. Load `main.lua` and start testing

4. Make your changes

5. Push and submit a PR!

## 📝 Commit Message Guidelines

Keep commits clear and descriptive:
- ✅ `Add Yuji's Domain Expansion move`
- ✅ `Fix combo counter not resetting properly`
- ✅ `Improve Black Flash timing window`
- ❌ `update stuff`
- ❌ `bug fix`

- Be respectful and constructive
- Help other contributors
- Share feedback and improvements
